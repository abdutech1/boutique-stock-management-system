# Boutique Stock Management System

## Overview
A stock and sales management system designed for boutique shops.
Products are managed by category and price-based subcategories instead of names.

## Key Features
- Category & price-based product registration
- Stock tracking with low-stock alerts
- Employee & owner sales tracking
- Automatic bonus calculation
- Weekly sales, salary, and profit reports

## Tech Stack
- Frontend: Next.js + TypeScript
- Backend: Node.js + Express + TypeScript
- Database: MySQL
- ORM: Prisma

## System Design
- ER Diagram (link or image)
- Clean separation of sales, stock, salaries, and expenses

## Business Rules
- Bonus is calculated automatically
- Stock is reduced on each sale
- Owner and employees can sell
- Low stock threshold: 3 items
